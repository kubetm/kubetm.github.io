1. Metrics Server 설치

1-1) Metrics Server 다운 및 설치

설치

git clone https://github.com/kubernetes-sigs/metrics-server.git
cd metrics-server
kubectl apply -f deploy/kubernetes/


Metrics Deployment 수정 (args 에 kubelet-insecure-tls 와 kubelet-preferred-address-types=InternalIP 추가)

kubectl edit deployment metrics-server -n kube-system

------------------------
spec:
   containers:
   - args:
     - --cert-dir=/tmp
     - --secure-port=4443
     - --kubelet-insecure-tls
     - --kubelet-preferred-address-types=InternalIP
     image: k8s.gcr.io/metrics-server-amd64:v0.3.6
     imagePullPolicy: IfNotPresent
     name: metrics-server
     ------------------------


설치 확인 (True값 확인)

kubectl get apiservices |egrep metrics

------------------------
v1beta1.metrics.k8s.io                     kube-system/metrics-server   True        28m
------------------------


메트릭 값 확인 (1~2분 후)

kubectl top node

2. HPA (Horizontal Pod Autoscaler)

2-1) Target Deployment / Service

apiVersion: apps/v1
kind: Deployment
metadata:
 name: stateless-app1
spec:
 selector:
   matchLabels:
      stateless: app
 replicas: 2
 template:
   metadata:
     labels:
       stateless: app
   spec:
     containers:
     - name: container
       image: tmkube/app:v1
       resources:
         requests:
           cpu: 200m
         limits:
           cpu: 500m
---
apiVersion: v1
kind: Service
metadata:
 name: stateless-svc1
spec:
 selector:
    stateless: app
 ports:
   - port: 8080
     targetPort: 8080
     nodePort: 30001
 type: NodePort


2-2) HPA - Resource

apiVersion: autoscaling/v2beta2
kind: HorizontalPodAutoscaler
metadata:
  name: hpa-resource
spec:
  maxReplicas: 10
  minReplicas: 2
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: stateless-app1
  metrics:
  - type: Resource 
    resource:
      name: cpu
      target:
        type: Utilization
        averageUtilization: 50

2-3) HPA - Pods

apiVersion: autoscaling/v2beta2
kind: HorizontalPodAutoscaler
metadata:
  name: hpa-pods
spec:
  maxReplicas: 10
  minReplicas: 2
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: stateless-app1
  metrics:
  - type: Pods
    pods:
      metric:
        name: packets-per-second
      target:
        type: AverageValue
        averageValue: 0.1k

2-4) HPA - Object

apiVersion: autoscaling/v2beta2
kind: HorizontalPodAutoscaler
metadata:
  name: hpa-object
spec:
  maxReplicas: 10
  minReplicas: 2
  scaleTargetRef:
    apiVersion: apps/v1
    kind: Deployment
    name: stateless-app1
  metrics:
  - type: Object
    object:
      metric:
        name: requests-per-second
      target:
        type: Value
        value: 0.1k
      describedObject:
        apiVersion: networking.k8s.io/v1beta1
        kind: Ingress
        name: ingress-hpa
