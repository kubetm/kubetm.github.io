[부록] Kubernetes 설치 개요

PPT 120% Github 90% Test 80% Recording 0%

테스트 중 현재 새버전에서 Dashboard 로그인에 대한 보안이 강화된게 확인 되었네요 ㅠ 
기존엔 보안없이 Skip 버튼으로 모든 기능을 쉽게 사용했었는데 지금 보니 인증을 필수로 사용하게 합니다. 
일단 최대한 쉽게 Dashboard를 사용할 수 있도록 고민을 해보고, 정 안되면 정석대로 항상 로그인해서 사용해야 겠네요. 
강의 예상 완료 일정보다 다소 지체될 수 있어 걱정이 됩니다. 그래도 빨리 만드는 것보다 제대로된 강의를 만들려고 하고요. 
지금으로선 2~3일 정도 더 지체될 수 있다는 점 말씀드릴께요. 기다리시는 분들께 죄송합니다.
